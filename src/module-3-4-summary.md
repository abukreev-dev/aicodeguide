# Module 3–4 — Параметры и настройки

## Фокус методички
Параметры — это скрытые рычаги управления поведением модели.

## Ключевые параметры

### Temperature
- Управляет степенью случайности.
- 0.1–0.3 — точность, аналитика.
- 0.7–1.2 — креатив и неожиданные идеи.
- Ошибка новичков: высокая температура в аналитических задачах.

### Top-p
- Управляет шириной словаря вариантов.
- Низкий — только самые вероятные слова.
- Высокий — редкие и нестандартные варианты.
- Меняет набор, а не хаотичность.

### Совместная работа
- Temperature = риск внутри набора.
- Top-p = размер набора.
- Низкие значения → строгость и предсказуемость.
- Высокие значения → максимальная креативность.

### Контекст как параметр
- Роль и стиль работают как управляющий рычаг.
- Без контекста параметры дают слабый эффект.

### Практические режимы
- Аналитика → низкая Temperature.
- Креатив → высокая Temperature.
- Надёжность → низкий Top-p.
- Разнообразие → высокий Top-p.

### Анти-паттерны
- Игнорирование параметров.
- Одновременное изменение всех настроек.
- Креативные параметры для задач на точность.
